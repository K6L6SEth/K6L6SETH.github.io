date: 2019-11-20 17:13:18
tags：

EVENT

这周信了老姐的邪，去清华听了一个佛学的讲座，顺便拜访了般若。名人不装暗逼，老子这周回了趟老家得到了家道刚刚开始兴盛时被坑而买的十分不值的房子。然而那也是一套房子啊，让我感觉如果以后混的不好也有个可以栖息的地方。在火车上看《龙族5》看的很爽，一下子攒了一大波，然而又在重要时刻卡章节，江南是tm真的狗。

讲座

讲座的人确实是个大佬，像在文献中经常见到的著名历史人物似的，一大堆帽子随便顶在头上，随便一个帽子对于普通人来说都是相当于皇冠的存在了。难以想象一位年轻的数学博士一本正经和你讲佛法。而且台下都是清华的禅学社成员听的一脸认真，我也只能配合表演。不过白嫖了两个杯子，我的位置对应的位置是“悟”我很喜欢。

OLD FRIEND

他看起来不太好。我发现我是一个主观性很强的人，我很讨厌承认这一点，喜欢一个人与否会极大的影响我的判断，而且这几乎是在见面时就注定的。但我的感觉却也一向很准。（真是悖论）。所以当他说他很讨厌学校线性代数用英文教材，并且还要强制学二外，校领导脑子都有坑的时候，我是十分相信他的。他还说知乎上的主流观点居然都力赞清华的时候，他感到“离谱”，像极了父亲的“荒唐”。他去了校心理咨询，医生建议他每周都去。我识趣的没有多问，仿佛他正在被内心的怒火折磨，让我想到了只狼里的怨恨之鬼。我们只有五站地的距离。

还和一起逃课的老朋友看了《天气之子》，感觉里面一些重要场景的音乐处理的相当好，《武士零》里主角打药的时候也有类似的滑轨变调处理，我十分喜欢。但是我仿佛已经过了电影的受众范围，因为我只觉得太理想，太中二了，两年前也许我会因此上头而对生活充满憧憬和希望，现在我只觉得男男女女烂俗而幼稚，唯一能让我上头除了游戏只有愤世嫉俗时。他还是老样子，日常逃课打游戏，我料定他不是需要学习的那种人。其实我和他性格上是一样的，只不过我没有成为混子的资格，我需要学习，我有责任。

房子

虽然只是名义上属于我，没有任何卵用，但我还是很高兴。我很想仔细研究研究，然后把它卖掉或出租，就有了启动资金抑或是每个月的小零花。总之每次一想到老子也有房子感觉糟糕的事情都不那么糟了。

龙族5

江南老狗的电视剧凉了，电影也凉了，只有安心写书了，终于也能做到一周三更了。然而记得初中看绝世唐门的时候tm唐三一天三更，啧啧，老狗啊老狗。虽然慢，但是总算有了点曾经的感觉，那种文笔的惊艳是让人看过一眼就无法忘掉的，虽然中二，但我总能记得。

可能是不知梦的缘故，流离之人追逐幻影。

凡王之血，必以剑终。 凡为过往，皆是序章。

可怕的不是收买灵魂的魔鬼，而是绝望的时候连可以交易的魔鬼都没有。

每个人能的心里都住着魔鬼。幸福是魔鬼的牢笼，当幸福化为泡影，魔鬼便会高唱着血腥的圣歌浮现，绝望之人将所向披靡。

男人们聚集在街边抽烟，一言不发，眼神也不交流，就像一群为不同君主宣誓效忠的武士，而吸烟点便是他们的中立区域，在飞散的烟雾中迷离的熄灭自己对朋友与亲人的思绪后，便踩灭烟头奔赴不同的未来。

看了龙族，有一件有浮世绘衬里的风衣也成为了我的人生梦想，去一趟日本秋叶原和在黄昏去四国爱媛县坐电车上山看风景也成了我的人生梦想。希望有一天可以实现。

说来惭愧又滑稽，我在高三才看的龙族，直接看到追更。感觉江南确实是经历过天堂也下过地狱的人，经历过纸醉金迷灯红酒绿（我所知道的大牌子全是小说里看的），但也被世界所遗忘，落魄街头闹市无人问，不然这种驾驭文字的能力不是这么简单就能获得的。我依然没有脱离龙族的受众，可能是因为主角路明非的丧与废和自己十分相像吧，总是那么的真实。

学业

高数第二类换元积分是tm真的sb。C语言学到指针了，这学期好像就要求到指针。非常有意思。vb也是真的sb，年纪轻轻就要受马旭平的苦，据说c++下学期也有可能是她，先来毒奶一波，一定要是她啊！！！

逐渐开始混了，大学宿舍生活真的会逐渐消磨一个人的意志，或者说我被iflab打败了，连般若都说中国3A不可能，我还有什么努力的理由？

创新杯也是真的sb，指导老师必须填但从来没指导过，好在还是有好老师真的存在的，可能这就是缘分吧。GET A LIFE的策划书我其实相当不满意，因为后期直接就变成我和3个混子的故事了，不过这也是我能找到的最好人选了。
